# Ekorafon

A marketplace platform connecting buyers and sellers across West Africa.

## Features

- AI-powered search
- Product discovery
- Merchant stores
- Secure payments
- Order management
- Customer support

## Technologies

- React
- Node.js
- MongoDB
- Express.js
- Paystack API
